## 📋 Project Overview
Microservices-based E-Commerce backend system with:
- **User Service** (Spring Boot + MySQL)
- **Product Service** (Spring Boot + MongoDB)
- **Order Service** (Spring Boot + PostgreSQL)
- **API Gateway** (Spring Cloud Gateway)
- **Gradio Frontend** (Python UI + Testing)
